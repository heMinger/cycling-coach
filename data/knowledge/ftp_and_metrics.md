# 核心训练指标

## FTP（功能性阈值功率，Functional Threshold Power）

**定义**：骑手在1小时内能维持的最大平均功率，单位瓦特（W）。
是所有功率训练区间的计算基础。

**测试方法**：
- 20分钟全力测试：取平均功率 × 0.95 = FTP
- 8分钟测试：取平均功率 × 0.90 = FTP
- Ramp Test：逐级递增功率至力竭，取最后完成阶段功率 × 0.75

**参考水平**（W/kg，功率体重比）：
| 水平 | 男性 | 女性 |
|------|------|------|
| 入门 | 2.0-2.5 | 1.5-2.0 |
| 业余 | 2.5-3.5 | 2.0-3.0 |
| 竞技 | 3.5-4.5 | 3.0-4.0 |
| 精英 | >4.5 | >4.0 |

FTP 建议每8-12周重新测试一次，以反映训练适应的进步。
训练期间 FTP 若提升 5% 以上，应及时更新所有区间功率设置。

---

## TSS（训练压力分，Training Stress Score）

**公式**：TSS = (时长秒 × NP × IF) / (FTP × 3600) × 100

**单次训练参考值**：
- <150：恢复快，次日可正常训练
- 150-300：中等疲劳，需1-2天恢复
- 300-450：高强度，需2-3天恢复
- >450：极高，需要数天充分恢复

**周TSS参考**：
- 恢复周：<300 TSS
- 一般训练周：400-600 TSS
- 高量积累周：600-900 TSS
- 赛前激活周：200-350 TSS

TSS 是量化训练负荷的核心工具，配合 CTL/ATL/TSB 使用可追踪体能与疲劳的长期趋势。
单独看一次训练的 TSS 意义有限，更重要的是看周、月维度的累积变化。

---

## IF（强度因子，Intensity Factor）

**公式**：IF = NP / FTP

**含义与典型值**：
- IF < 0.65：极轻松恢复骑
- IF 0.65-0.75：轻松长距离骑行（Z2）
- IF 0.75-0.85：节奏训练（Z3）
- IF 0.85-0.95：竞速/阈值训练（Z4）
- IF 0.95-1.05：接近或在 FTP 附近，用于阈值间歇
- IF > 1.05：短时间高强度冲刺，无法长时间维持

IF 是判断一次训练强度分布的快速指标。同样时长的训练，IF 越高则 TSS 越高、疲劳越大。
长距离耐力骑行 IF 通常控制在 0.65-0.75 之间以保证可持续性。

---

## NP（标准化功率，Normalized Power）

**概念**：NP 通过四次方加权平均算法，将功率波动对身体的实际代谢消耗标准化。
比平均功率更能反映骑行的真实训练负荷。

**为什么 NP 比平均功率更重要**：
起伏路面或间歇训练中，功率剧烈波动。生理学研究表明高功率段的代谢消耗远超低功率段，
因此简单平均功率会低估实际生理负荷。NP 修正了这一偏差。

**典型场景**：
- 平坦路面匀速骑行：NP ≈ 平均功率（波动小）
- 山地/起伏路面：NP 通常比平均功率高 10-20%
- 冲刺+恢复的间歇训练：NP 可比平均功率高 20-30%

NP 是计算 IF 和 TSS 的基础，使用平均功率计算会严重低估训练压力。

---

## EF（效率因子，Efficiency Factor）

**公式**：EF = NP / 平均心率

**含义**：EF 反映心肺效率——同等心率输出下能产生多少功率。
EF 越高，说明心肺系统越高效，有氧能力越强。

**如何使用 EF**：
- 在相似条件下（温度、路线、疲劳度）对比不同时期的 EF 值
- EF 随 Z2 训练积累稳步上升，是有氧基础建设的客观指标
- 高温、疲劳或咖啡因摄入会导致心率偏高，此时 EF 偏低，需排除干扰因素

**局限性**：EF 不适合跨越不同骑行类型（如爬坡 vs 平路）直接比较，
只在条件控制相近的同类训练间横向对比才有意义。

---

## HRV（心率变异性，Heart Rate Variability）

**定义**：相邻心跳间隔时间的变化程度，反映自主神经系统平衡状态，间接衡量身体恢复程度。

**测量方法**：每日晨起（起床前静躺）测量3-5分钟，使用专用 HRV 应用记录趋势。

**解读**：
- HRV 相对个人基准值下降 ≥20%：建议当日减量或休息
- HRV 持续偏低（连续3天以上）：过度训练预警，需主动恢复
- HRV 高于基准：身体恢复良好，可执行计划内高强度训练

**影响 HRV 的因素**：
- 降低 HRV：高强度训练后、睡眠不足、酒精、生病、高压力
- 提升 HRV：充足睡眠、有氧基础训练、冥想/深呼吸练习

HRV 是主观感觉的客观补充，结合 TSB 和主观疲劳感共同判断训练可行性。
